## Usage

To use this `Taskfile`, include it in your project's `Taskfile`. For example:

```yaml
version: '3'

includes:
  trufflehog: https://raw.githubusercontent.com/vanyda-official/taskfile-trufflehog/main/tasks.yaml
```
